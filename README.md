# origami-crease-pattern-editor

折痕图绘制工具

## 启动

```
# Install dependencies
npm install

# Run in `debug` mode, to debug app using VSCODE
npm run debug

# Run in `dev` mode
npm run dev

# Build installer for this app
npm run build
```
